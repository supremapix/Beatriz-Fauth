## 1. MELHORIAS NO HEADER E HERO DA HOME

### Header (Navegação Superior)
**Objetivo:** Criar um header mais delicado e minimalista

- **Design do Header:**
  - Reduzir altura do header para 70-80px com fundo branco puro ou off-white suave (#FAFAFA)

  - Menu hamburguer em mobile com animação suave de fade-in

  - **Subtítulo:** "Psicanálise e psiquiatria em Balneário Camboriú"

  - Cor: cinza médio (#5A5A5A)
  - Espaçamento de 16px abaixo do título

- **Cores e Paleta:**
  - **Principal:** Verde-azulado suave #5B8C8E (traz calma e confiança)
  - **Secundária:** Rosa pó suave #E8D5D5 (acolhimento feminino)

### Animações para Seções ao Scroll
```javascript
// Intersection Observer para animações ao scroll
const observerOptions = {
  threshold: 0.1,
  rootMargin: '0px 0px -100px 0px'
};

const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.classList.add('animate-in');
    }
  });
}, observerOptions);

// Aplicar em todas as seções
document.querySelectorAll('section').forEach(section => {
  observer.observe(section);
});
```

```css
/* Classe para elementos antes de animar */
section {
  opacity: 0;
  transform: translateY(30px);
  transition: all 0.8s cubic-bezier(0.4, 0, 0.2, 1);
}

section.animate-in {
  opacity: 1;
  transform: translateY(0);
}
```

// JavaScript para mostrar/ocultar botão
const backToTopBtn = document.getElementById('backToTop');

window.addEventListener('scroll', () => {
  if (window.scrollY > 400) {
    backToTopBtn.classList.add('visible');
  } else {
    backToTopBtn.classList.remove('visible');
  }
});

backToTopBtn.addEventListener('click', () => {

    behavior: 'smooth'
  });

### Melhorias Gerais
1. **Espaçamento e Respiração:**
   - Aumentar padding das seções (100-120px vertical)
   - Espaçamento entre elementos (24-32px)
   - Max-width do conteúdo: 1200px com margens laterais generosas

2. **Tipografia Melhorada:**
   - Títulos de seção: 36-42px (mobile: 28-32px)
   - Subtítulos: 24-28px
   - Texto corpo: 16-18px com line-height: 1.7
   - Hierarquia clara com pesos variados (300, 400, 600)

3. **Estados Interativos:**
   - Todos os links com hover suave
   - Botões com estados :hover, :active, :focus bem definidos

   - Loading states em formulários
   - Mensagens de sucesso/erro com cores apropriadas
   - Indicadores de campo obrigatório claros

5. **Acessibilidade:**
   - Contraste adequado (WCAG AA mínimo)
   - Aria-labels em todos os botões de ícone
   - Foco visível em navegação por teclado

## INSTRUÇÕES FINAIS PARA O REPLIT AGENT

Por favor, implemente estas melhorias mantendo:
1. **Delicadeza visual** - cores suaves, transições suaves, sem elementos agressivos

Priorize as seções Hero, Header e CTAs principais primeiro, depois as melhorias de UX e animações complementares.
